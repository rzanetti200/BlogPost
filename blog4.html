Variables can also be stored in vanilla CSS. You can assign a CSS value to a variable with the var() method. this is helpfull if you dont want to have to keep refereing to a specific value each and every time you code in css.

:root {
    --big-size: 100px;
}

.var {
    font-size: var(--big-size);
}

<p>I will make this <span class="var">word</span> very big!</p>

I attempted to learn some coding shortcuts that would allow me to code more quickly. The first technique I discovered was that I could simply type the element without its tags and VSCode will take care of the rest. For example, if I type "p" or "div," VSCode will autocomplete it with "p" or "div." <p> <p/> or <div> <div/>.
Another example of a shortcut is the ability to type an element followed by a . and then you can type out anything, and that will become the name of the class for that element.
An example of this would look like div.classname which would turn into <div class="classname"> <div/>
The final trick I learned was adding a * followed by a number before declarding an element and it will create the number amount of elements.
This would look like div*5, and would create this:

<div> <div/>
<div> <div/>
<div> <div/>
<div> <div/>
<div> <div/>
