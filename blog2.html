Something i stumbled across this week got me excited. i learned how to change the color of links when someone hoveres over them or clicks on them. This helps the user know that the link is clickable and makes my website seem more custom and professional.

here is some sample code demonstrating how this would happen

            A:link { COLOR: black; TEXT-DECORATION: none; font-weight: normal }

            A:active { COLOR: black; TEXT-DECORATION: none }
