I read an article which describes 10 important html tags that every one should know. After reading this article I have a better understanding of how to write code more efficently. One important tag that anyone working with html should know is the <cite>, this tag allows the user to define text inside of the element as a reference. This tag works like the <blackquote> element, except that it will also make the word italic this can be easily changed using some simple css if needed. The <cite>, tag is useful for citing bibliographic and other cite refrences.

I also found the <address> tag can also be useful, in allowing you to semantically markup addresses in HTML. It also allows the user to italicize all the data that is given to it. As i've written more code in html i've found that it's difficult to always know what revisions you have made in your code this is where tags like <ins> and <del> come in. These tags will cross out anything in the <del> brackets and <ins> on the other hand will show the user to user the revision made by underlining the word inside the <ins> brackets.
