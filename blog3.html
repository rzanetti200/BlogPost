I learned how to make a css animation. I can specify how long the animation lasts, how many repetitions there are, and how the animation transitions without javascript by using a number of css elements.

animation!
@keyframes fade-anim {
    from {background-color: red;}
    to {background-color: black;}
}

#anim-block {
    padding: 20px;
    background-color: #ffffff;
    animation-name: fade-anim;
    animation-duration: 3s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
}

This week, I also learned about another interesting HTML element: the <details> tag.
It appears to be a very handy tag for when a developer has a lot of information that can be expanded on but would take up too much space otherwise. For example, rather than presenting all of the information on all of the different breeds of cats at once, I could hide it until a user wishes to see a single cat's information.
There are numerous additional applications for this, such as different sorts of cars or anything with a lot of distinct types of information.
