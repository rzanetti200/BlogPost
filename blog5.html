I just saw an article that outlined ten essential HTML tags that everyone should be familiar with. I now have a better knowledge of how to create code more effectively as a result of reading this post. Anyone dealing with HTML should be familiar with the <cite> tag, which allows the user to designate text inside the element as a reference. This tag functions similarly to the <blackquote> element, with the exception that it also makes the word italic. This can be readily altered with some simple CSS if necessary. The tag <cite> is used for quoting bibliographic and other referenced sources.

I discovered that the <address> tag can also be useful for semantically marking up addresses in HTML. It also allows the user to italicize all of the information provided to it. As I've written more html code, I've discovered that it's tough to keep track of the changes you've made to your code. This is where tags like <ins> and <del> come in handy. These tags will cross out anything inside the <del> brackets, while <ins> will highlight the word inside the <ins> brackets to show the user the revision made.
