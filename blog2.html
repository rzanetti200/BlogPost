HTML Coding Shortcuts

I tried to learn some shortcuts that would help me speed up my coding. The first trick I learned was that I could just type out the element without its tags and VSCode would do the rest for me. For example I could type in "p" or "div" and VSCode would auto complete it to <p> <p/> or <div> <div/>.
Another example of a shortcut is the ability to type an element followed by a . and then you can type out anything, and that will become the name of the class for that element.
An example of this would look like div.classname which would turn into <div class="classname"> <div/>
The final trick I learned was adding a * followed by a number before declarding an element and it will create the number amount of elements.
